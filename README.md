# Quest Diagnostics (quest-diagnostics)

Quest Diagnostics is a Fortune 500 company and the world's leading provider of diagnostic information services. This repository captures the APIs, integration products, and machine-readable artifacts known for Quest Diagnostics. Quest does not publish a public developer portal; integrations are typically delivered through their Quanum family of products.

## APIs

### Quanum Data Exchange
Quanum Data Exchange is Quest's interoperability service for delivering laboratory and diagnostic results into electronic health record systems, enabling integrated ordering and resulting workflows across healthcare organizations.

**Human URL:** [https://www.questdiagnostics.com/business-solutions/health-systems-it-solutions](https://www.questdiagnostics.com/business-solutions/health-systems-it-solutions)

### Quanum eLabs
Quanum eLabs is a web-based laboratory ordering and results delivery service that integrates with provider workflows so clinicians can submit orders, track samples, and review Quest results electronically.

**Human URL:** [https://www.questdiagnostics.com/healthcare-professionals/about-our-tests/quanum-elabs](https://www.questdiagnostics.com/healthcare-professionals/about-our-tests/quanum-elabs)
